# PCB Designs using KiCad

A collection of PCB design projects created using **KiCad** for embedded systems and electronics applications.

## Projects

### 01. Arduino LED PCB
- Arduino Uno LED interface PCB
- Single LED output with current-limiting resistor
- Beginner PCB project

### 02. 7805 Power Supply PCB
- Regulated 5V power supply
- Based on the LM7805 voltage regulator
- Suitable for embedded systems

### 03. ESP32 + DHT11 PCB
- ESP32 development board interface
- Temperature & Humidity sensor
- Compact PCB layout
